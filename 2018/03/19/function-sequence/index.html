函数列

Posted on 2018-03-19 | Edited on 2018-09-30 | In math

函数列

  1. 定义:元素为函数而非数的数列

函数列与函数级数的关系

  1. 任意给定一个函数列\(\{f_n(x)\}\), 对\(f_n(x)\), 可以通过构造\(u_n(x) = f_n(x) - f_{n-1}(x)\)来形成新的函数列,那么级数 \(\displaystyle\sum_n^\infty u_n(x)\) = 函数列元素\(f_n(x)\).
  2. 由此, 对级数收敛性的讨论可以转变成对函数列\(f_n(x)\)收敛性的讨论.
  3. 函数列与数列具有相似的部分,可以使用数列收敛性的讨论方法来进行.
  4. 同时函数具有的连续性也会带来其特殊的微分学性质(连续,可微,可积)

函数列的收敛意味着什么

  1. 类比数列, \(\{a_n\}\)收敛意味着\(\lim\limits_{n\to \infty} a_n = C\), 而针对函数列则是\(\lim\limits_{n\to\infty}f_n(x) = f(x)\)
  2. f(x)可以是多样化的, 如分段函数1, 而f(x)的定义域\(D\)取决于收敛域(参考 函数项级数.md)
  3. 把函数看成一种具有函数特征的一种特殊的数

函数列的收敛

收敛的方式

  1. 逐点收敛:
    1. 来源: 取某一点x0, 验证函数列 \(\{f_n(x)\}\) 在该点所退化形成的数列 \(\{f_n(x_0)\} (\{a_n\})\)是否收敛至一常数 \(C\), 逐一验证, 得到由所有收敛点形成的集合D及它们所对应的值形成的集合A, 以及对应关系 \(f\), 形成映射 \(f: D \to A\), 该映射记作 \(f(x)\), 即为函数列逐点收敛得到的收敛函数.
    2. 特点:
      1. 逐一验证即逐一求极限, 用 \(\epsilon-N\)语句所描绘的\(N\)可能不同
      2. \(N \backsim (\epsilon, x)\) 与x有关
      3. 不同点的收敛速度不同,可能差距很大, N越小收敛速度越快.
  2. 一致收敛:
    1. 来源: \(\epsilon-N\)的 \(N\)取收敛判定的max{\(N_i\)}
    2. 特点:
      1. \(N\) 相同
      2. \(N \backsim (\epsilon)\) 与x无关
      3. 收敛速度呈现某种整体性
    3. 记号:
      • 记作 \(f_n(x)\rightrightarrows f(x) (n \to \infty)\)
    4. 判别:
      • 是否存在 $max(N_i) $(N是否有界)2
      • 结合N越大收敛速度越慢可得知, 要一致收敛, 所有收敛都必须快于某一速度
        • 每一收敛都存在一个最小N
        • 对极限为C的数列来说,收敛到A(A 不等于 C)的速度为无限慢3

一致收敛的判别定理

  1. 确界判别的充要条件:

    • 内容:
      > \(\lim\limits_{x \to \infty }\sup\limits_{x \in D}|f_n(x) - f(x)| = 0\), 则 \(\{f_n(x)\}\)在\(D\)上一致收敛于\(f\)

    • 解释:
      利用lim 中的 $-N $语言体现了存在一个有界的 \(N\)
    • 运用:
      1. 证明 \({x^n}\)不一致收敛4
      2. 意味着收敛速度的一致
    • 例题:
      1. 判定 \(f_n(x) = (1-x)x^n, x\in[0,1]\)的一致收敛性
        > 解答: \(\sup\limits_{x\in[0,1]}|(1-x)x^n|\)的求得应该把n当为常数对待, 当 \(x = \frac{n}{n+1}\)时, 取得 \(\sup\)
  2. 比较判别的充要条件:

    • 内容: > \(f_n(x)\text{收敛于}f(x), x \in D, and \, a_n\to0\), 若\(\forall x\in D, |f_n(x) - f(x)| \le a_n\), 则一致收敛

    • 解释:
      一致收敛的函数对象相同

    • 运用:
      对已知逐点收敛的目标收敛函数,可以用放缩法来简化对一致收敛的判定

  3. 归结原理判别法
    1. 本质: 数列可以看成函数的子列,即满足收敛的子列判别法
    2. 内容:
      > $f_nf(n), xD {x_n}, x_n D, st._{n}|f_n(x_n) - f(x_n)| = 0 $

  4. Cauchy一致收敛准则

    1. 内容:
      > \(\forall \epsilon > 0, \, \exists N = N(\epsilon), \forall m, n> N, \forall x\in D: |f_m(x) - f_n(x)| < \epsilon\)

    2. 本质: 取的\(N \backsim \epsilon\)

收敛函数列的性质

  1. 连续性
    1. 内容: \(\text{If} \, f_n(x) \rightrightarrows f, \,x\in D, \text{and} \, \text{each} \, f_n(x) \text{连续}, then f \, \text{连续} x\in D\)

    2. 本质:

      1. 各项连续, 由于一致收敛的收敛速度一致性, 导致其连续性的趋向一致, 若收敛函数\(f\)不连续, 即存在一断点, 这一断点会导致某收敛速度的无限慢5, 实例6

      2. 极限可交换性:
        > \(\lim\limits_{x\to x0}\lim\limits_{n\to\infty}f_n(x) = \lim\limits_{n\to\infty}\lim\limits_{x\to x_0}f_n(x)\)

        解释: 左项先取极限函数f, 再取x0对应函数值; 右项先取收敛点x0, 退化为常数列, 再取常数列的收敛极限.

  2. 可积性

  1. 如\(\{f_n(x)\}\)的通项\(f_n(x) = x^n\), 则\(\lim\limits_{n\to\infty}f_n(x) = \begin{cases}0, & -1<x<1 \\ 1, & x=1\end{cases}\), \(f(x)\)为一分段函数,但该函数显然不连续↩

  2. 如 \(f_n(x) = x^n, \, x \in (-1, 1)\). 显然 \(f_n(x) \to 0, \, x \in (-1, 1)\). 但若要使 $> 0, st. n > N, |f_n(x) - 0| < $, 那么 \(|x^n| < \epsilon\), 即 \(n > ln\epsilon/lnx\), 当 \(x \to 1\)时, \(ln\epsilon/x \to \infty\), \(N\) 无界, 所以不存在 \(max\{N(\epsilon, x)\}\)(x=1时收敛速度非常慢)↩

  3. 无限慢的概念基于速度与 \(N\)的关系, 也就是N越大, 收敛速度越慢, N不存在[^N不存在]时可看作无限慢↩

  4. \(\lim\limits_{n\to\infty}\sup\limits_{x\in(-1, 1)}|x^n-0|=1 > 0 (\lim\limits_{x\to1}x^n = 1)\)↩

  5. 无限慢的概念基于速度与 \(N\)的关系, 也就是N越大, 收敛速度越慢, N不存在[^N不存在]时可看作无限慢↩

  6. 如 \(\{x^n\}, \, x\in [0, 1]\), 其在各项均连续,但其收敛函数则是一个在1点处间断的分段函数, 在1点的收敛到0的速度 由此在区间[0, 1]上不一致收敛至0.↩
